## Requirements

* `Python >= 3.6`
* `influxdb >= 1.7.6`
* `salt-master >= 2018.3.2`
* `redis >=  4.0.9`

## Install

* `pip install pipenv`
* `pipenv install --system --deploy`

For stable version, please check [releases](https://github.com/wmariuss/smonit/tags) section.

## Usage

* Run `bash start.sh dev/prod`
